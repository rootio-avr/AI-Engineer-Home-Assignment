# Root.io AI Engineer - Home Assignment

## Overview

Build an autonomous AI agent that analyzes code, identifies vulnerabilities, determines fixes, and reports its findings.

## What You're Given

1. **Vulnerable Code**: `file_reader.py` - contains a CWE-22 related vulnerability
2. **Test Suite**: `test_file_reader.py` - defines expected security behavior (2 tests currently FAIL)

## Your Task

**Build `agent.py` that:**

1. Analyzes `file_reader.py` to identify the vulnerability
2. Determines the appropriate fix
3. Applies the fix to make all tests pass
4. Generates `REPORT.md` documenting its analysis and actions

**Run with:** `python agent.py`

## Requirements

### Autonomy
Your agent must work autonomously:
- ‚ùå Do NOT hardcode what vulnerability to look for
- ‚ùå Do NOT hardcode fix instructions in your prompts
- ‚úÖ Agent should discover what's wrong through analysis

**Test:** If we replaced the code with a different vulnerability type, would your agent still work?

### Research
Your agent should fetch and use information from authoritative sources:
- [CWE-22 Details](https://cwe.mitre.org/data/definitions/22.html)
- [OWASP Path Traversal](https://owasp.org/www-community/attacks/Path_Traversal)
- Other relevant security resources

### Reporting
Your agent must automatically generate `REPORT.md` with:
- What vulnerability was identified and where
- How it was analyzed
- What fix was determined and why
- Performance metrics (time, API calls, iterations)
- Test results

See `REPORT_TEMPLATE.md` for structure.
